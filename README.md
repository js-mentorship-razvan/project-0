# project-0

This app is designed for Spin & Go games. It's purpose is to make preflop decision easier for players and create a profitable strategy.

This chart allows you to select any given position and any given stack size to make the best readless preflop decision. 

Making the right decision preflop can save you many chips in game by avoiding uncomfortable spots.
